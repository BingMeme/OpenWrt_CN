# 在openwrt终端使用tcpdump程序抓包        

## 安装tcpdump     
1. root身份进入openwrt终端，先执行同步软件源命令：`opkg update`
2. 再执行安装命令`opkg install tcpdump`
3. 查看安装版本命令：`tcpdump --help`有显示就证明安装成功了
